### Installation

Requires [Node](https://nodejs.org/) v16.15.0 to run.
Requires [NPM](https://www.npmjs.com/) v8.10.0 to run.
Requires [YARN](https://yarnpkg.com/) v1.22.18 to run.

### Frontend

With yarn

```sh
$ cd ccxc-test-project
$ yarn
$ yarn dev
```

With npm

```sh
$ cd ccxc-test-project
$ npm install
$ npm run dev
```
